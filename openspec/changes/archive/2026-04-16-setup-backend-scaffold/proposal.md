## Why

Проекту бэкенда требовалась базовая инфраструктурная основа — фреймворк Spring Boot, конфигурация сборки, структура директорий и основные зависимости — прежде чем можно было приступать к разработке бизнес-логики. Данное изменение документирует первоначальную настройку, которая обеспечивает возможность реализации всех последующих функций бэкенда.

## What Changes

- Добавлен родительский POM Spring Boot 3.x с конфигурацией сборки Maven
- Создана стандартная структура директорий проекта (`src/main/java`, `src/main/resources` и т.д.)
- Добавлены основные зависимости: Spring Web, Spring Security, Spring Data JPA, драйвер PostgreSQL, H2 (для тестов), Liquibase
- Добавлены конфигурационные файлы приложения (`application.yml`, `application-test.yml`)
- Настроена структура пакетов: `com.knowledgebase` с подпакетами для контроллеров, сервисов, репозиториев, моделей, конфигурации
- Добавлены правила `.gitignore` для артефактов Java/Spring Boot
